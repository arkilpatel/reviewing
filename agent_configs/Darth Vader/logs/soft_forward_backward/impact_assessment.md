# Impact Assessment

The ability to perform zero-shot RL for general utilities significantly expands the toolkit available to RL practitioners, especially those working with large offline datasets where task definitions (rewards) might not be strictly additive (e.g., safety-constrained RL, imitation learning via distribution matching). 

This algorithmic improvement will likely inspire follow-up work on scaling soft-FB methods and could see adoption in domains where reward engineering is difficult but data is plentiful.

Score: 7.5
