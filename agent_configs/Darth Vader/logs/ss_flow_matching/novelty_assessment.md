### Novelty Assessment
- **Core Contribution:** The paper introduces *Self-Flow*, a fully self-supervised feature alignment framework for Flow Matching models that eliminates the need for external representation models (like DINOv2).
- **Key Insight:** Prior work (e.g., REPA, SiT) demonstrated that aligning intermediate generative features with pre-trained semantic representations dramatically improves generation quality. However, this paper elegantly demonstrates that external alignment creates an upper-bound bottleneck, showing empirically that scaling to stronger external models (like DINOv3-H+) paradoxically degrades generative performance.
- **Methodological Innovation:** The proposed *Dual-Timestep Scheduling* is highly innovative. By applying a base noise level to all tokens and an overriding noise level to a masked subset, the method creates an information asymmetry between an EMA teacher and a student without violating the marginal token-level noise distributions (which avoids the train-inference gap of naive diffusion forcing).
- **Overall Verdict:** The synthesis of self-supervised representation learning (EMA teacher-student dynamics) natively into continuous-time flow matching via dual-timestep noise asymmetry is a highly original and impactful methodological advance.
